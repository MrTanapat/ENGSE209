## 1. ระบบนี้ทำงานอะไรได้บ้างในปัจจุบัน?

ปัจจุบันระบบมี 5 menu ประกอบด้วย

- **Show All:** แสดงข้อมูลรายการสินค้า ID, Name, Stock, Price, Type
  - ตัวอย่าง: `ID: 101 | Name: Mama Noodles | Stock: 50 | Price: 6.0 THB | Type: Food`
- **Add or Update:** เพิ่มข้อมูลสินค้าเข้าไปในระบบ ID, Name, Stock, Price, Type
- **Out:** นำสินค้าใน Stock ออกเป็นจำนวนโดยเลือก ID ก่อนที่จะนำสินค้าใน Stock ออก และมีแจ้งเตือนสินค้าต่ำกว่า 5 ชิ้น
- **Check Check:** แสดงภาพรวมของสินค้า Total product, Total Inventory, แจ้งเตือนสินค้าต่ำกว่า 10 ชิ้น
- **Exit:** จบการทำงานระบบ

## 3. จุดไหนในโค้ดที่อ่านแล้วรู้สึก "งง" หรือคิดว่าเป็นจุดเสี่ยงที่สุด?

1. **ไม่มี Input Validation** ถ้า Input `"abc"` เข้าไปโปรแกรมจะ Crash ทันที
2. **ทุกอย่างอยู่ในไฟล์เดียวกัน** ปัจจุบันยังไม่มีปัญหา แต่ถ้าระบบมีหลายฟีเจอร์มากขึ้น อาจแก้ไขหรือปรับปรุงได้ยาก
3. **`x = {}`** เป็น Global Variable สามารถแก้ไขได้จากหลายฟังก์ชัน ทำให้ติดตามการเปลี่ยนแปลงข้อมูลได้ยาก อาจเกิดการแก้ไขข้อมูลโดยไม่ตั้งใจส่งผลให้เกิดข้อผิดพลาดได้
